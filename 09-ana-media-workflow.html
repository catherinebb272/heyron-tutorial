Module 09 — ANA Media Workflow

⏱️ 10 minutes
📋 Workflow Guide
This guide walks through setting up the media creation workflow for Apart Not Alone (ANA) — using ElevenLabs for voice and LipSync AI for animation.

The Workflow

This module covers how to use your AI agent to create audio and animated content for your sobriety milestone coins. The workflow involves:

🎯 Complete Media Workflow

  • Step 1: Generate voice audio using ElevenLabs TTS
  • Step 2: Use the voice audio with LipSync AI to create animated avatar videos
  • Step 3: Export and use in your marketing/content

Step 1: ElevenLabs Voice Generation

Your agent can generate text-to-speech using your ElevenLabs API key. Here's what you need to know:

Setup Required

Before using voice generation, you'll need to provide your ElevenLabs API key. The agent stores this securely in the workspace environment.

💡 Important: Free Tier Limitation
ElevenLabs free tier has a limit on API calls. Rapid requests trigger "unusual activity" detection and may disable the free tier. Space out your requests or consider upgrading to a paid plan for production use.

Sample Prompt for Voice Generation

Prompt to Use
"We need to set up ElevenLabs to generate a voice sample for our otter avatar. Generate a 5 second test sample using [voice name] with the script: [your script text here]."
This prompt gets pasted into ChatGPT or other image processor to get the desired image. Feed the same image back into ChatGPT to get the same character rendered in different ways.

If Eleven Labs isn't configured

"Then return to the prompt above to get the audio file."

API Notes

🔧 Technical Details

  • API Key Location: Stored in `.env` file as `ELEVENLABS_API_KEY`
  • Voice ID Retrieval: Can be retrieved via `GET /v1/voices` API call
  • Model: Uses `eleven_turbo_v2` for speed

Step 2: LipSync AI Animation

Once you have your voice audio, use LipSync AI to animate your avatar image.

Platform

lipSync.video — No sign-up required for free tier

What You'll Need Ready

  • 📷 Your otter avatar image (PNG/JPG, front-facing works best)
  • 🎵 A 5-second audio file (mp3/wav) with the voice script

Sample Prompt for LipSync Setup

Prompt to Use

"We need to set up LipSync AI to create a sample animation. Talk me through creating a free 5 second animation of our [otter] avatar using the image and voice script we already created."

Step-by-Step Process

  1. Go to https://lipsync.video/
  2. Click "Try AI Lip Sync" — no account needed for basic use
  3. Upload your otter image — front-facing with clear mouth area works best
  4. Add audio: Upload your pre-recorded voice file (mp3/wav)
  5. Set duration to 5 seconds
  6. Click "Generate" — AI creates the lip-synced animation
  7. Download your video
📝 Note
The platform works with still images — it animates the mouth to match your audio. For best results, use a clear image where the mouth area is distinct.

Summary

This workflow enables you to:

  • Generate voiceovers using ElevenLabs TTS with your API
  • Animate your avatar using LipSync AI
  • Create engaging content for your ANA milestone coins

The agent can handle the ElevenLabs generation directly. LipSync AI requires your interaction through their web interface.
